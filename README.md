# Northstar Admin Dashboard

A polished, responsive React admin dashboard built for portfolio and client demo use.

## Live Project Summary
This project demonstrates a modern admin experience with:
- Responsive sidebar layout for desktop and mobile
- Light/Dark mode with local theme persistence
- Reusable dashboard widgets (stats, charts, data table)
- Users, Analytics, Settings, Support, and Logout screens
- Clean UI system with consistent spacing, typography, and color tokens

## Tech Stack
- React 19
- Vite 7
- Tailwind CSS 4
- Recharts

## Screens Included
- Dashboard overview
- Users management
- Analytics trends
- Settings and preferences
- Support center with ticket form
- Logout confirmation page

## Run Locally
```bash
npm install
npm run dev
```

## Build for Production
```bash
npm run build
npm run preview
```
